## Git Commands Practiced
The following commands were used to manage this project:
* `git init`: Initializes a new Git repository.
* `git status`: Shows the state of the working directory and staging area.
* `git add`: Adds file contents to the staging area.
* `git commit`: Saves the staged snapshot to the project history.
* `git remote add`: Links the local repository to a remote GitHub server.
* `git push`: Uploads local repository content to the remote repository.
